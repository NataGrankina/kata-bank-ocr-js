Bank OCR Kata
=============
My JavaScript solution to the [KataBankOCR](http://codingdojo.org/cgi-bin/index.pl?KataBankOCR).

TRY IT OUT
==========
Sample input files are located in ```test/example_input```.

For example:

```
$ bin/process_accounts.js test/example_input/accounts.txt
```

RUNNING TESTS
=============
```
$ npm install -g mocha
```

Run the tests with:

```
$ mocha test
```
